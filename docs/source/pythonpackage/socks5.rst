Socks5 module
==============

socks5man.socks5
-----------------

This module contains the Socks5 class. It is used as a wrapper to contain socks5
server information. The wrapper allows you to verify if a socks5 server is operational, calculate an approximate
bandwidth, calculate an approximate connection time, and retrieve all stored server information through properties.
